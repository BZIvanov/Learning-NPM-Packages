Customers

List all customers

GET /customers

Details:

  • You can find your customers by clicking Customers on the left panel in the Stripe dashboard

Create customer

POST /customers

{
  "email": "some.user@mail.com",
  "description": "my first customer"
}

Delete customer

DELETE /customers/:id

Products

List all products

GET /products

Details:

  • You can find your products by clicking Products on the left panel in the Stripe dashboard

Create product

POST /products

{
  "name": "White handbag"
}

Delete product

DELETE /products/:id

Prices

List all prices

GET /prices

Details:

  • You can find your products by clicking Products on the left panel in the Stripe dashboard

Create price

POST /prices

{
  "productId": "prod_J79ENxF7F2waCO"
}

Details:

  • A price is always attached to a product, so you should provide existing product ID.
  • You can delete a price in the Dashboard by navigating to the product details. You can't delete prices which were used in transactions, only archive them.

Payment methods

Get payment method

GET /payment-methods/:paymentId

Create payment method

POST /payment-methods

Details:

  • Creates payment method of type 'card'

Attach payment method to a customer

POST /payment-methods/:paymentId

{
  "customerId": "cus_JvYTcuCbxVeluF"
}

Subscriptions

List all subscriptions

GET /subscriptions

Create payment method

POST /subscriptions

{
  "customerId": "cus_JvYTcuCbxVeluF",
  "priceId": "price_1IUz4QCE2xwH3UTcjI87II7m"
}

Details:

  • Creates subscription for chosen customer
